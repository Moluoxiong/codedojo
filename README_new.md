# 📚 CodeDojo 文档索引

欢迎来到 **CodeDojo 项目文档中心**。此目录包含项目从设计、架构到教学记录的所有资料。

---

## 📁 文档结构

| 文件/文件夹 | 说明 |
| :--- | :--- |
| `ARCHITECTURE.md` | 系统架构说明（前后端分层图） |
| `setup_guide.md` | 环境搭建指南（新手快速开始） |
| `novice_journal.md` | 学习日志与问题解决记录 |
| `PRD/` | 产品需求文档目录 |
| `README.md` | 本索引页 |

---

## 🧭 使用说明

1. 按时间阅读 `novice_journal.md` 了解开发历程；  
2. 查阅 `setup_guide.md` 以快速配置开发环境；  
3. 根据 `ARCHITECTURE.md` 理解系统整体逻辑；  
4. 若要了解产品功能计划，请阅读 `PRD/` 下各阶段文档。
